# 更新日志

本文档记录项目的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2025-01-XX

### 新增

#### 核心功能
- 命令管理功能
  - 创建、编辑、删除命令
  - 命令排序（拖拽排序）
  - 命令复制功能
  - 命令导入/导出（JSON 格式）

#### 命令执行
- 命令执行功能
  - 执行命令并查看实时日志
  - 停止正在执行的命令
  - 命令执行状态监控（idle、running、success、failed）
  - 命令执行历史记录

#### 系统集成
- 系统托盘支持
  - 最小化到系统托盘
  - 托盘图标菜单
  - 从托盘恢复窗口
- 开机自启动功能（Windows）
- 系统信息监控
  - CPU 使用率监控
  - 内存使用率监控
  - 磁盘使用率监控

#### 用户界面
- 现代化 UI 设计
  - 基于 Ant Design 6 的界面
  - 深色模式支持
  - 响应式布局
- 多语言支持
  - 简体中文
  - 英文
- 命令日志查看页面
  - 实时日志输出
  - 日志滚动显示
  - 命令状态显示

#### 配置管理
- 系统配置页面
  - 语言设置
  - 开机自启动设置
- 配置持久化存储（SQLite）

#### 开发体验
- TypeScript 类型支持
- 错误边界处理
- 日志系统
- 单元测试框架（Vitest）

### 技术栈

#### 前端
- React 19
- TypeScript
- Ant Design 6
- Styled Components
- React Router 7
- i18next

#### 后端
- Rust
- Tauri 2
- SQLite (rusqlite)
- sysinfo

### 安全
- CSP（Content Security Policy）配置
- 生产环境禁用开发者工具
- 安全的命令执行机制
